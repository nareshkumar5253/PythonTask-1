# PythonTask-1

This repository contains simple Python programs that demonstrate basic Python concepts such as printing, variables, operators, indexing, and more.

## Tasks Included

### Task 1 – Print Formatting
Uses `sep` and `end` parameters in the `print()` function to format output.




### Task 2 – Variables
Demonstrates creating variables and printing them with a separator.


### Task 3 – Multiple Assignment
Shows assigning multiple values to multiple variables in a single line.



### Task 4 – String Indexing
Accesses characters from a string using positive and negative indexing.



### Task 5 – Arithmetic Operators
Demonstrates basic arithmetic operations.

Operators used:
- Addition `+`
- Subtraction `-`
- Multiplication `*`
- Division `/`
- Modulus `%`
- Exponent `**`
- Floor Division `//`

### Task 6 – BODMAS Expression
Shows operator precedence in Python.



### Task 7 – Assignment Operators
Uses compound assignment operators like `+=` and `/=`.

### Task 8 – Comparison Operators
Compares values and returns `True` or `False`.

Operators used:
- `>`
- `<`
- `==`
- `!=`
- `>=`
- `<=`

### Task 9 – String Comparison
Compares two strings and shows case sensitivity.


### Task 10 – Logical Operators
Demonstrates logical operators:

- `and`
- `or`
- `not`

### Task 11 – Membership Operator
Checks if an element exists in a list.

Operators used:
- `in`
- `not in`

### Task 12 – Swap Variables
Swaps two variables without using a temporary variable.



### Task 13 – Bitwise XOR
Performs a bitwise XOR operation.
